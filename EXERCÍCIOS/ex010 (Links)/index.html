Usando Links

Links Externos

Com a tag <a> você pode adicionar um link ao seu texto
Exemplo:

Aqui você pode conferir todos os volumes de Nana

Colocar a tag target="_blank" dentro da tag <a> faz com que o link seja aberto em outra aba
Exemplo:

Aqui você pode conferir todos os volumes de Nana

Também é indicado usar a tag "rel="xxx" dentro da tag <a>, para indicar quando o link é:

  • external (Externo)
  • author (Do mesmo autor do documento atual)
  • prev (A parte anterior do documento atual)
  • next (A parte seguinte do documento atual)
  • nofollow (Não dou aval ao conteúdo)

Links Internos

Você também pode adicionar um link interno ao seu texto, caso a página esteja na mesma pasta do index
Exemplo:

Essa é minha segunda página

Você também pode adicionar um link interno de uma sub-pasta ao seu texto, caso a pasta esteja na pasta do index, usando subpasta/
Exemplo:

Essa é minha terceira página

Links de Download

Para adicionar um link de download, o arquivo deve estar na mesma ou em uma sub-pasta do index, colocando as tags download="arquivo" e type="xxx" o link procede a fazer o download
Exemplo

  • Baixar NANA 21 em PDF
  • Baixar NANA 21 em ZIP
