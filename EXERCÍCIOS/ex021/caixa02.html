Dicas para montar o site

"header" para cabeçalho, dentro de "header" podemos colocar a tags "nav" para navegação (colocando os links dentro dela)
"main" para o corpo do site, dentro de "main" podemos colocar a tag "section" e dividir as "section" por id's (id="XXX")
"footer" para o pé do site
