Código-fonte

A tag <code> serve para deixar um texto em uma fonte monoespaçada lateralmente

Exemplo:

O comando document.getElementById("teste") é escrito em liguagem JavaScript

Se usar apenas a tag <code> o texto fica todo reto, desorganizado

num = int(input('Digite o número')) if num % 2 == 0: print(f'0 número {num} é PAR') else print(f'0 número {num} é ÍMPAR') print('fim do programa')

Se usar a tag <pre> antes da tag <code> o texto fica organizado do jeito que foi escrito

        
        num = int(input('Digite o número'))
    if num % 2 == 0:
        print(f'0 número {num} é PAR')
    else
        print(f'0 número {num} é ÍMPAR')
    print('fim do programa')

Para mexer texto horizoantalmente dentro do <pre> selecionar o texto e apertar:
Tab para cado espaço que for mexer para a direita
Shift+Tab para cado espaço que for mexer para a esquerda

Citações simples

A tag <q> serve para marcar uma citação na mesma linha

Exemplo:

Como diria o pai de um amigo: o computador é um burro muito rápido

Citações completas

A tag <blockquote> serve para marcar uma citação completa

Exemplo:

Segundo KNITEMAREX a respeito de Soul Eater:

Soul Eater is an impressive anime. It's one of those where you will no doubt enjoy watching.

A tag <blockquote cite="link do site de onde saiu a quote"> faz com que a citação mostre a fonte de onde foi retirada

Exemplo:

Segundo KNITEMAREX a respeito de Soul Eater:

Soul Eater is an impressive anime. It's one of those where you will no doubt enjoy watching.

Abreviações

A tag <abbr title="xxx"> serve para que o cursor mostre o significado da abreviação quando passa por cima

Exemplo:

Eu adoro SE, ID e DB.

Texto invertido

A tag <bdo dir="rtl"> serve para escrever textos invertidos

Exemplo:

Vou assistir Soul Eater(Vou assistir Soul Eater)
