Trabalhando com Listas

Listas Ordenadas

A tag <ol> cria uma lista ordenada, cada item deve ser colocado com a tag <li> dentro da tag <ol>
Exemplo:

  1. Blackstar
  2. Crona
  3. Genya
  4. Trunks

Com a tag "type" dentro da tag <ol>, podemos mudar o tipo de marcação da lista, tendo como opções: 1 (padrão), A, a, I e i
Exemplo:

  1. Blackstar
  2. Crona
  3. Genya
  4. Trunks

Ou:

  1. Blackstar
  2. Crona
  3. Genya
  4. Trunks

Com a tag "start" dentro da tag <ol>, podemos mudar o número onde a lista começa
Exemplo:

  1. Blackstar
  2. Crona
  3. Genya
  4. Trunks

Listas Não Ordenadas

A tag <ul> cria uma lista desordenada, cada item deve ser colocado com a tag <li> dentro da tag <ul>
Exemplo:

  • Blackstar
  • Crona
  • Genya
  • Trunks

Com a tag "type" dentro da tag <ul>, podemos mudar o tipo de marcação da lista, tendo como opções: disc (padrão), circle e square
Exemplo:

  • Blackstar
  • Crona
  • Genya
  • Trunks

Lista de Definições

A tag <dl> cria uma lista de definição, cada item deve ser colocado com a tag <dt> para indicar o termo/item, e a tag <dd> para indicar a definição dentro da tag <dl>
Exemplo:

HTML
Linguagem para criação do conteúdo de um site
CSS
Linguagem de marcação para crição do design de um site
JavaScript
Linguagem de programação para a criação da interatividade de um site
